# gatsby-source-trackingmore

Gatsby source plugin for fetching carriers from Trackingmore.com

## Install

```bash
yarn add gatsby-source-trackingmore
```

## How to use

```js
// In your gatsby-config.js
plugins: [
  {
    resolve: "gatsby-source-trackingmore",
    options: {
      apiKey: "...",
    },
  },
];
```
